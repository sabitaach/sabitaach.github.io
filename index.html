About

I am a PhD candidate at the University of Illinois at Chicago (UIC). I am a member of the Natural Language Processing lab at UIC and my advisor is Prof. Barbara Di Eugenio . I am generally interested in the practical applications of Machine Learning and Natural Language Processing techniques. My PhD thesis is on generating personalized hospital-stay summaries for patients.
I received my B.S degree in Computer Engineering from Tribhuvan University, Institute of Engineering in 2011.

    • Research Projects

    Ongoing

    PatientNarr: This research involves using Machine Learning and Natural Language Generation techniques for making it easier for patients to understand the content of their health documents. This research comprises of three main tasks: multi-document abstractive summarization, simplification, and personalization.

    Previous

    ChiQat: ChiQat is an educational software for helping undergraduate students in understanding basic Computer Science concepts. I designed a quiz module for assessing the knowledge of the user before and after using the system.

    Course Projects (selected)

  • Using Evolutionary Programming and Supervised Learning for Connect Four: Implemented neural networks using genetic algorithm and backpropagation to play the game of Connect Four. (KeyWords: Neural Networks, Language: Matlab)
  • Tell Me How to Cook: Developed a mixed initiative spoken dialogue system that can have conversation with the user and assist in cooking by giving recipe details and providing companionship. (KeyWords: Dialogue System, Language: Java)
  • Predicting Bike Sharing Demand: Predicted bike requirements based on several factors like weather, temperature, day, hour and season using linear and non-linear regression techniques. (KeyWords: Machine Learning; Language: Java, R, Matlab)
  • Question Analysis in Travel Domain Question Answering system: Developed a system that assigns appropriate labels to the questions posed by the user depending on the expected answer type. (KeyWords: Natural Language Processing, Database; Language: Java)
  • Content Selection from Semantic Web Data: Developed a system that can identify the sentences from Wikipedia that contain the information represented in the RDF triple. (KeyWords: Data and Web Semantics, Linked Open Data Cloud; Language: Java, RDF)
  • Ontology Matching in Agreement-Maker: Developed a matching algorithm for Agreement-Maker for Ontology Alignment Evaluation Initiative (OAEI) 2014 Conference track. (KeyWords: Data and Web Semantics, Ontology; Language: Java, RDF and OWL)
  • Geospatial data model for Data Explorer of Geospatial data Integration, Visualization and Analytics (GIVA): Developed an ontology that could model any kind of geospatial data for GIVA. (KeyWords: Ontologies, Protégé; Language: OWL)
  • Sentiment analysis for cosmetic product reviews: Developed a system to analyze the collective sentiment of cosmetic product reviews. (KeyWords: Data Mining, Language: Java)
  • Natural Language Interface for Database: Developed an interface for natural language queries for Winter Olympics. Ranked 3rd among all submissions. (KeyWords: Natural Language Processing, Database; Language: Java)
